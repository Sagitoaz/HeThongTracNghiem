# codechay/

Đây là thư mục để mỗi thành viên tự code phiên bản **siêu đơn giản** của các trang được phân công.

## Cách dùng

1. Tạo thư mục riêng theo tên: `tv1/`, `tv2/`, `tv3/`, `tv4/`
2. Code HTML + CSS + JS thuần, không cần framework
3. Dữ liệu fix cứng trong JS hoặc dùng `localStorage`
4. Xem phân công chi tiết tại [docs/phan-cong-nhom.md](../docs/phan-cong-nhom.md)
